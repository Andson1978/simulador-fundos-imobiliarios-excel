# Simulador de Investimentos em Fundos Imobiliários (Excel)

## Descrição
Este projeto consiste em um simulador de investimentos em Fundos Imobiliários (FIIs),
desenvolvido em Microsoft Excel, com o objetivo de auxiliar investidores na análise de
aportes, rendimentos mensais e projeção de patrimônio ao longo do tempo.

## Funcionalidades
- Simulação de aportes mensais
- Cálculo de rendimentos com base em dividend yield
- Projeção de patrimônio acumulado
- Estimativa de renda passiva mensal
- Interface simples e intuitiva

## Como utilizar
1. Baixe o arquivo `Simulador_FII.xlsx`
2. Abra no Microsoft Excel
3. Preencha os campos de aporte, taxa de rendimento e período
4. Analise os resultados gerados automaticamente
